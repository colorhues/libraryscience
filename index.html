An experiment in two parts Un experimento en dos partes

This site documents what it's like to learn library science as it's practiced in a primary-school educational setting — and to experiment, in parallel, with how AI tools can support the healthy execution of that work. Este sitio documenta cómo es aprender ciencias bibliotecarias tal como se practican en el entorno educativo de una escuela primaria — y experimentar, en paralelo, con cómo las herramientas de IA pueden apoyar la ejecución saludable de ese trabajo.

The Updates section carries unfiltered field notes from the experience. The Tools section indexes the small utilities built along the way to make library work go a little smoother for the people doing it. La sección de Actualizaciones contiene notas de campo sin filtrar de la experiencia. La sección de Herramientas indexa las pequeñas utilidades construidas en el camino para hacer que el trabajo bibliotecario sea un poco más fluido para las personas que lo realizan.

MARC File Creator

Scan an ISBN and a local barcode; the app pulls bibliographic data from Open Library and Google Books and writes a Destiny-ready MARC 21 record. Outputs a .mrc file ready to import. Escanea un ISBN y un código de barras local; la aplicación extrae datos bibliográficos de Open Library y Google Books y genera un registro MARC 21 listo para Destiny. Produce un archivo .mrc listo para importar.
