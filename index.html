002 · About

An experiment in two parts

This site documents what it's like to learn library science as it's practiced in a primary-school educational setting — and to experiment, in parallel, with how AI tools can support the healthy execution of that work.

The Updates section carries unfiltered field notes from the experience. The Tools section indexes the small utilities built along the way to make library work go a little smoother for the people doing it.

MARC File Creator

Scan an ISBN and a local barcode; the app pulls bibliographic data from Open Library and Google Books and writes a Destiny-ready MARC 21 record. Outputs a .mrc file ready to import.

Bibi · Volunteer Scheduling

Chat with Bibi to find a time to come help at the library — read to kids, scan books, sort and label, or help plan an author visit. Bibi confirms a slot conversationally and sends a Google Calendar invite with the address and check-in details.
